# Flex-box-Task

## Project Description

This project is a simple and responsive web page that displays a list of blog articles about Jimmy Choo perfumes. The page is designed using CSS Grid and Flexbox, and is used to display products (articles) as beautiful cards. The page content is in Persian, and the Vazirmatn font is used for better text display.

## Features

- **Responsive**: The page displays well on different devices (mobile, tablet, desktop).
- **Using Flexbox and Grid**: CSS Flexbox and Grid are used for layout and alignment of elements.
- **Persian Font**: Vazirmatn font is used for proper display of Persian text.
- **Product Cards**: Each article is displayed as a card that includes an image, title, description, date, and read link.
- **Responsive Grid**: Cards are displayed in different numbers of columns (1 to 4) based on screen size.

## Technologies

- **HTML5**: Page structure
- **CSS3**: Styling and layout (Flexbox, Grid, Media Queries)
- **Vazirmatn Font**: Persian font for text

## How to Run

1. Download or clone the project files.
2. Open the `index.html` file in a web browser.
3. The page will run locally.

No server or special tools are needed, as this is a static page.
